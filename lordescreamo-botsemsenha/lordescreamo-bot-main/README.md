## Install
Follow The Steps Below!

```bash
> termux-setup-storage
(after that tap on permission)
> apt install git -y
> git clone https://github.com/Carlos-a11y/carlos.git
> cd termux-whatsapp-bot
> bash install.sh
```

## Usage

```bash
> npm start
```
